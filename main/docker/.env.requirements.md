# Environment Variable Requirements for main/docker

This file documents all variables required by main/docker/docker-compose.yml.

| Variable | Required | Format | Default in .env_example | Description |
|---|---|---|---|---|
| POSTGRES_IMAGE | yes | docker image tag | postgres:15 | PostgreSQL container image |
| TRINO_IMAGE | yes | docker image tag | trinodb/trino:444 | Trino container image |
| POSTGRES_USER | yes | string | pvlakehouse | PostgreSQL username |
| POSTGRES_PASSWORD | yes | string (secret) | change_me_postgres_password | PostgreSQL password |
| POSTGRES_DB | yes | string | pvlakehouse | Default PostgreSQL database |
| POSTGRES_PORT | yes | integer port | 5432 | Host port mapped to PostgreSQL 5432 |
| TRINO_PORT | yes | integer port | 8081 | Host port mapped to Trino 8080 |

## Notes
- Keep real secrets only in main/docker/.env.
- Commit main/docker/.env_example and this requirements file.
- Replace default passwords before shared or production-like usage.
