# Environment Variable Requirements for dev/config (Solar AI Chat)

| Variable | Required | Format | Default in .env_example | Description |
|---|---|---|---|---|
| SOLAR_CHAT_GEMINI_API_KEY | yes | string (secret) | replace_with_google_gemini_api_key | API key used for Gemini API calls |
| SOLAR_CHAT_GEMINI_BASE_URL | yes | URL | https://generativelanguage.googleapis.com/v1beta | Base URL for Gemini API endpoint |
| SOLAR_CHAT_PRIMARY_MODEL | yes | model id | gemini-2.5-flash-lite | Primary Gemini model |
| SOLAR_CHAT_FALLBACK_MODEL | yes | model id | gemini-2.5-flash | Fallback Gemini model |
| SOLAR_CHAT_REQUEST_TIMEOUT_SECONDS | yes | float seconds | 3.2 | Timeout target to keep responses under 4 seconds for standard queries |
| SOLAR_CHAT_DATA_ROOT | no | absolute or relative path | empty | Optional override path for Silver/Gold CSV sources |

## Notes
- Keep real secrets only in `dev/config/.env` and do not commit that file.
- Keep `.env_example` and `.env.requirements.md` committed for reproducible setup.
- Ensure role and model logs are enabled in your runtime logger configuration.
